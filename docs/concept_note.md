# 企画書・仕様書：2Dアクションゲームプロジェクト（仮題：Chaos Gear）

## 1. エレベーターピッチ

「10秒ごとに操作が反転する」という理不尽なカオスを、プレイヤースキルと探索による能力拡張でねじ伏せる、実況映え特化型のメトロイドヴァニア・2Dハードコアアクション。

## 2. コアメカニクス（10秒ギミック）

* **基本ルール:** 10秒経過するごとに、特定のアクションボタンの割り当てが強制的に入れ替わる。
* **グリッチ・ボーナス:** 操作切り替え直後（0.5秒以内）にジャストタイミングで正しい操作を入力すると、自機に強力なバフ（無敵時間や攻撃力UP）が付与され、ピンチをチャンスに変える爽快感を生む。
* **UIによる補助（視線移動ゼロUI）:** 画面上にコントローラーは表示せず、主人公の周囲に「ホログラム・リング」を展開。現在のボタン割り当てと、エフェクトの色で直感的に操作状態を伝達する。
* **正常化（オアシス）:** マップ各所に配置された「休憩スポット」に到達すると、書き換えられた操作状況がリセットされ、初期の正常な操作に戻る。

## 3. レベルデザインと進行

* **形式:** メトロイドヴァニア型（探索型の1つの巨大なマップ）。
* **探索と解放:** ボス討伐や探索によって装備（二段ジャンプ、新必殺技など）を拡張し、それまで行けなかったエリアを開拓していく。
* **デスペナルティ:** HPゲージ制。HPがゼロになったらゲームオーバーとなり、直近でアクセスした「休憩スポット」から再開される（『Hollow Knight』形式）。

## 4. プレイヤーのアクション全容

* **左右移動:** 地上・空中ともに可能。多少の慣性は残しつつもキビキビ動くレスポンスを重視。
* **ジャンプ:** ボタンの長押し時間により「小ジャンプ」「中ジャンプ」に分岐。強化により「空中ジャンプ（二段ジャンプ）」が解放される。
* **通常攻撃（射撃）:** 遠距離射撃。チャージ可能。8方向への撃ち分けが可能で、ジャンプ中の下向き射撃などアクロバティックな攻撃にも対応。
* **必殺攻撃:** 専用ゲージを消費して発動。強化要素（アンロック）として複数種類存在し、状況に応じて選択・切り替えが可能。
* **ダッシュ:** 短距離を高速移動する。強化により「無敵時間（回避アクション）」が付与される。

## 5. 世界観・キャラクターデザイン

* **主人公:** 廃棄寸前のボロボロなロボット。四角い頭部、青く光る目、口元を隠すマントが特徴。自己修復機能の暴走により配線が書き換わってしまう設定。
* **敵・ボスキャラクター:** 主人公とは対照的な、最新鋭の高精細ロボット群。四神（白虎、青龍、朱雀、玄武）をモチーフにした近未来サイボーグや、多関節の怪獣/エイリアン型巨大ロボットなど。
* **オープニング演出:** ゲームエンジン内（インエンジン）でのスプライトアニメーションとタイマー制御によるシームレスなカットシーンを採用。
